🎯 What requirements did you achieve?

(Answer)
The Message Board: I built a social network where you can write messages, change them, or delete them.

The Log-In Key: I made it so the app knows who you are. This keeps your messages safe so only you can edit them.

The Ring System: Every time you post a message or get a like, you earn a Ring. Your total rings show up at the top of the screen! You also earn your first ring when you register your profile details. You are unable to post a message until this has been done.

Special Glow: If a message gets 10 likes, it gets a cool glowing border to show it is a popular post.

Profile Pictures: I set it up so your personal picture shows up next to everything you post.
